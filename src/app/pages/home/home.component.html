¡Bienvenid@!

Este proyecto está hecho por Sergio Sierra. Es una versión demo de una aplicación contable realizada en Angular 14.2.3

¿Qué es este el proyecto?

Es la versión demo de una aplicación contable hecha en Angular, la cual de forma muy intuitiva hace que llevar tus cuentas personales sea más fácil que nunca. todos los datos son ficticios, no está conectada a una base de datos pero se puede llegar a operar con las cuentas y los movimientos para ver su comportamiento.

¿Qué tecnología he usado?

  • HTML personalizado
  • CSS y responsive design
  • Framework Angular
  • Iconos Fontawesome

¿Qué funciones tiene?

  • Inicio de sesión
  • Anotar movimientos
  • Gestión de cuentas
  • Balance final
