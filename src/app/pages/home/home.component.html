Esta web es una versión demo , realizada por Sergio Sierra se trata de una aplicación para la gestion de las finanzas personales.

¿Qué tecnología he usado?

  • HTML personalizo
  • CSS y responsive design
  • Framework Angular
  • Iconos Fontawesome

¿Qué funciones tiene?

  • Inicio de sesión
  • Anotar movimientos
  • Gestión de cuentas
  • Balance final

¿Qué es el proyecto?

Es una versión demo en la que los todos los datos son ficticios, no está conectada a una base de datos pero se puede llegar a operar con las cuentas y los movimientos para ver su comportamiento, todo gracias a los componentes de Angular.
